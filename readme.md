# Estudos de React e JavaScript

Repositório com atividades e microprojetos desenvolvidos durante meus estudos de React e JavaScript.

## 📚 Conteúdo

* Hooks (useState, useEffect, etc.)
* Componentização
* Consumo de APIs
* Manipulação de estado
* Exercícios práticos

## 🎯 Objetivo

Consolidar fundamentos do React e JavaScript através de prática constante.

## ⚙️ Como executar

Cada pasta representa um projeto ou exercício independente.

Para rodar:

```bash
npm install
npm start
```

## 📝 Observações

* Projetos focados em aprendizado (não produção)
* Código pode evoluir conforme avanço nos estudos
